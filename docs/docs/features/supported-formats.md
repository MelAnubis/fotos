# Supported formats

Immich supports a number of image and video formats, which are outlined here.

## Image formats

| Format            | Supported?           | Notes                                            |
| :---------------- | :----------------:   | :----------------------------------------------- |
| `IMMICH_VERSION`  | :white_check_mark:   | `release`                                        |
| `No`              | :x:                  |                                                  |

| `No`              | :x:                  |                                                  |
